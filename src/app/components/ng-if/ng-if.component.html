*ngIf Directive Implementation

Property "display":

Direct use - *ngIf

Text displayed when the checkbox is "activated" - true

Text displayed when the checkbox is "deactivated" - false

Using *ngIf - else (ng-template)

Text displayed when the checkbox is "activated" - true

Text displayed when the checkbox is "deactivated" - false

Using *ngIf - then - else

Text displayed when the checkbox is "activated" - true Text displayed when the checkbox is "deactivated" - false

Using hidden attribute

Text displayed when the checkbox is "deactivated" - false

Text displayed when when the checkbox is "activated" - true
